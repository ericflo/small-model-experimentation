# Decision Records

Decision records capture strategic pivots that affect more than one experiment. They are not required for ordinary runs; use them when the repository changes direction, creates or retires a program, adopts a new shared benchmark, or decides not to repeat a tempting line.

Each record should include:

- the decision,
- the prior evidence,
- alternatives considered,
- what would reverse the decision,
- links to affected programs, experiments, and claim pages.

Use [templates/decision_record.md](../../templates/decision_record.md) for new records.
