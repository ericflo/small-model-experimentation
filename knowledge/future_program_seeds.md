# Future Program Seeds

These are candidate research lines that may deserve their own programs after one or two scoped probes. They are intentionally not forced into the current registry yet.

## Multimodal And Embodied Small Models

Question: do the structured execution and evidence-selection patterns survive when inputs include images, UI state, audio, or physical observations?

First probe: adapt one executable-intermediate experiment to a simple visual table or UI transformation task.

## Data Generation And Synthetic Curriculum Design

Question: can the repository learn which synthetic curricula create transfer rather than benchmark-specific competence?

First probe: compare human-designed, model-generated, and failure-mined curricula on one held-out primitive suite.

## Small-Model Collaboration

Question: can multiple small models with different roles outperform one larger or single-policy small model under fixed budget?

First probe: assign generator, verifier, critic, and evidence-acquirer roles on an existing candidate pool.

## On-Device And Latency-Constrained Agents

Question: which mechanisms preserve gains when inference budget, memory, and tool calls are severely constrained?

First probe: rerun a selector/tool-control experiment with strict latency and token ceilings.

## Human-Agent Research Interfaces

Question: what interfaces let people and agents inspect, branch, and critique experiment evidence fastest?

First probe: build a static report over programs, claims, and open questions, then measure whether a new agent finds relevant prior work faster.
