- `repo_tasks.py`: ten fresh constrained repository families with host-only partial/oracle states.
- `repo_agent.py`: looping JSON-tool coding agent and controlled recovery starts.
- `bank.py`: executable patch mining and exact conditional transition calibration.
- `harness.py` / `vllm_runner.py`: experiment-local pinned Qwen3.5-4B inference path with merged-checkpoint support.
