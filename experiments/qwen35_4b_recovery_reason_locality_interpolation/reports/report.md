## Summary

**Status: preregistered; result-bearing run pending.** This follow-up preserves

## Research Program Fit

The parent established a useful action-learning endpoint and a sharp plan-dose
failure, but stopped before any unseen repository family. This experiment asks
whether weight interpolation can extract the plan arm's valid, concise recovery
without paying its unrelated-language collateral, then subjects one frozen
point to the transfer and matched-compute tests the parent never opened.

## Method

Four predeclared action→reason mixtures are merged from the same apex checkpoint
in float32. The full ladder is screened on unrelated contexts with logit drift,
entropy, and varentropy; only passing points reach the known train-family
selection block. One winner receives an independent locality confirmation and,
only then, two fresh family-held-out coding blocks.
