## Novelty Claim

First curriculum in the line designed from quantified failure forensics of its
own predecessors' raw outputs: staged answer formats, demonstrated bounded
search, oversampled hardest cases (early bugs; co-located injections), and a
prospectively documented answer-normalization fix for a measured grader
artifact.

## Calibration and kill rule

This is the axis line's third content iteration. Believability: each dead
block's failure mechanism is precisely known and each fix targets it directly;
the installed blocks are untouched. Kill rule: if the redesigned repair
lessons still fail to install (no bugfind/bugmend holdout wins over both
controls), the trace-repair axis closes for this model at this dose and the
line pivots rather than iterating a v3 of the same mechanism.

## Mechanism

The v1 model faithfully executed the taught prefix and improvised where the
lesson hand-waved. V2 removes the hand-wave: every think demonstrates the
candidate tests it expects the model to run, including the rewrite classes the
truth requires and the model never proposed, with explicit termination. The
explanation is false if the demonstrated search still does not transfer to
held-out tasks at matched exposure against the replay control.
