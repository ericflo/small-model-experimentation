# Source

- `task_data.py`: copied self-contained exact-depth generator with a new seed
  block and 322 fresh task rows.
- `model_ops.py`: pinned cached native traces, close-only free-form replay, and
  exact cache-free constrained plus unmasked slot logits.
- `scripts/run.py`: immutable/data/power hashes, task-bootstrap/diversity gates,
  exact shuffle/cardinality contracts, qualification/confirmation, and
  fail-closed later stages.
- `scripts/power_audit.py`: parent-task planning receipt; actual decisions use
  the nonparametric bootstrap in `run.py`.
