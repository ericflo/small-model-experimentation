## Novelty Claim

This is the first universal-curriculum trial to join variable-depth natural-language
state transitions, independently recomputed hypothesis scores, and a verified
answer-only commit in one truth-audited training interface.

## Mechanism

Natural-language procedures should elicit the model's stronger simulation mode while
an explicit table prevents state from drifting across depth. Scoring every candidate
hypothesis on every probe prevents premature branch collapse. A final verification
row followed by a deliberately short answer trains the exact seam the predecessor
missed. The explanation is false if these lessons do not improve fresh execution and
hypothesis selection over both the parent and same-compute replay, or if gains require
copying the table vocabulary at evaluation.

## Control Plan

- Baseline: authenticated `close_xi` parent.
- Mechanism-falsifying control: independent same-parent replay continuation, exactly
  matched on encoded forward tokens and optimizer steps, with a shared byte-identical
  replay core in the same positions.
- Shift or robustness check: fresh experiment-owned local procedures use unseen
  surfaces, variable depths, and ordinary natural-language instructions without the
  training table labels.
- Hidden-label boundary: benchmarks remain read-forbidden. Only their public family
  contract and aggregate gateway may be invoked, conditionally after local promotion.
