# Idea Funnel

Use this funnel when deciding whether a new idea belongs in the repository and where it should land.

## 1. Prior Evidence

Find the closest program in `research_programs/` and at least three relevant experiments through `knowledge/research_program_index.md` or `knowledge/experiment_catalog.md`.

## 2. Novel Uncertainty

Write the uncertainty as a single sentence. If it is already answered by prior evidence, do not run the experiment; add a synthesis note instead.

## 3. Program Fit

Choose one:

- Attach to an existing program.
- Create a new program.
- Mark as exploratory only if it is a small probe and explain where it might land later.

Use `make new-program` or `make new-experiment` once the fit is clear.

## 4. Control Plan

Name the baseline and the control that would most directly falsify the mechanism.

## 5. Evidence Output

Decide what the experiment will add to the knowledgebase:

- a confirmed claim,
- a negative result,
- a new failure mode,
- a reusable artifact,
- a new follow-up branch.
